# HF × skeletal-muscle ageing — shared transcriptional signature

A reproducible, hypothesis-generating bioinformatics pipeline mapping a shared
transcriptional signature between **heart failure (HF)** left-ventricular myocardium
and **ageing skeletal muscle**, with a CLIP-supported upstream miRNA/TF network and
single-cell localisation.

> **Honest framing:** this is a *hypothesis-generating in-silico map of a candidate
> cardio-muscle ageing axis*, not a validated diagnostic biomarker. The HF arm of
> the signature is robust to age and etiology; the muscle arm remains ageing-linked.
> See `results/PEER_REVIEW_AND_ROBUSTNESS.md`.

## Key results
- 59 shared dysregulated genes (hypergeometric p = 1.0×10⁻³)
- 8-gene directional signature: HF external **AUC 0.92**; ageing muscle 5-cohort
  meta **AUC 0.76** (95% CI 0.61–0.86, I² = 0%)
- HF component age-robust (AUC 0.95 after age adjustment) and etiology-robust
- Candidate axis: **miR-15/16/195/497 → CCND1**; NAMPT→myeloid, FBLN1→stroma (scRNA)

## Layout
- `scripts/` — numbered pipeline `00`–`17` + `run_all.R` driver (deterministic, seed 42)
- `results/` — figures, tables, `RESULTS_SUMMARY.md`, `PEER_REVIEW_AND_ROBUSTNESS.md`
- `MANUSCRIPT.md` — full manuscript draft
- `submission/` — Genes & Genomics formatted manuscript + cover letter + figures

## Reproduce
```bash
MODE=cross Rscript scripts/run_all.R            # steps 01–08, 10
RUN_SCRNA=1 MODE=cross Rscript scripts/run_all.R   # + step 09 (single-cell, heavy)
```
Raw GEO/STRING inputs (`data/`) and large `.rds` intermediates are git-ignored and
re-fetched/regenerated by step 01.
